Banc de test du projet

Banc de test de stabilité, localisé dans le même bureau que mon poste de travail. Le PC est un "streamer" (injecteur DVB). Une seule fréquence (soit quelques chaines) est effectivement diffusée. Les quatre STBs zappent entre ces quelques chaines toutes les 10s. Ce banc à permis d'investiguer des bugs rares (race conditions), des fuites mémoires, etc. Ou simplement de tester la version courante par rapport aux indicateurs de performance souhaités (KPI - Key Performance Indicators).
