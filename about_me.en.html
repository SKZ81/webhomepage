ABOUT ME

Hobbies

Right now I'm mainly in QuadCopter building.

I choose the hard way about the software : while I ordered an hardware frame nearly ready-to-fly,

I started to build my own Flight Controller all by myself, starting from an Arduino hardware (I mean, using those 8bit Atmel's AVR CPU as the main CPU).


Other implied components are Inertia Measurement Units (gyroscope, acceleroscope, magnetoscope), and a pression sensor as an altimeter.

Flying in 8bits is not a goal by itself, but creating a robot controller using an AVR was one.

I am currently implementing One Shot 125 as an output signal from the flight controller i'm intending to build

I am usig simulators (Atmel Studio, SimulAvr, simavr) and i'm trying to find out a way to automize non-regression test using one of them.
